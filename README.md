# 📈 KSE-100 Stock Close Price Predictor

A Machine Learning project that predicts the **Closing Price** of 
Pakistan's KSE-100 Stock Exchange using historical market data.

## 🔍 What's Inside
- Data Preprocessing & Cleaning
- Exploratory Data Analysis (EDA) with visualizations
- 3 ML Models: Linear Regression, Decision Tree, Random Forest
- Model Evaluation & Comparison (MAE, MSE, R² Score)
- Interactive Streamlit Web App for live predictions

## 🏆 Best Model: Linear Regression
- R² Score: 0.9998
- MAE: 48.16

## 🛠️ Tools Used
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Streamlit

## 📊 Dataset
Pakistan Stock Exchange — KSE-100 Index (historical OHLCV data)
